# My Tools Web App

学習用に作成した、シンプルなWebツール集です。

---

## 文字数カウンター  
入力したテキストの文字数をリアルタイムに表示します。

**機能**
- テキスト入力欄
- 入力文字数を即時カウント
- レイアウト中央配置

---

## パスワード生成ツール  
指定した文字数と条件に応じて、ランダムなパスワードを作成します。

**機能**
- パスワードの文字数を指定
- 英字(大文字・小文字)・記号の追加可
- コピー機能
